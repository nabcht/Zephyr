## Overview
Zephyr should feel like a high-end terminal for AI: minimalist, precise, and professional.

## Colors
- Use Primary for headers and sidebars.
- Use Accent specifically for run or execute buttons and highlighted status chips.
- Use Background for the main chat or dashboard workspace.
