## Overview
uZephyr should feel like a high-end terminal for AI: minimalist, precise, and professional.

## Colors
- Use Primary (#1A1C1E) for headers and sidebars.
- Use Accent (#B8422E) specifically for run or execute buttons and highlighted status chips.
- Use Background (#F7F5F2) for the main chat or dashboard workspace.
